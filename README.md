# WeMCP

## Quick start

Please ensure the following conditions are met before installing WeMCP:

- CPU >= 4 Virtual Cores
- RAM >= 8 GB
- Docker 19.03.9+
- Docker Compose 1.25.1+

### Download WeMCP

```shell
git clone https://github.com/tomeai/wemcp.git
sudo docker compose up -d
```
